# ASP.NET Core Identity Authentication

A custom user authentication system built with ASP.NET Core MVC using Identity Framework.

## Features
- User Registration & Login
- Password Reset (Forgot Password)
- Secure Password Handling
- Identity-based Authentication & Authorization

## Tech Stack
- ASP.NET Core MVC
- ASP.NET Core Identity
- Entity Framework Core
- SQL Server

## Purpose
Demonstrates real-world authentication workflows using ASP.NET Core Identity.

## Getting Started
1. Clone the repository
2. Update the database connection string
3. Run migrations
4. Start the application
